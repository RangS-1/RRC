Left Sidebar Folder

v1.0.0

This section is organized under a separate folder structure (docs/navbar-kiri/) to demonstrate nested document routing in the left sidebar menu.

Folder Structure

By splitting docs pages into standalone folders, you achieve cleaner URLs (e.g. /docs/navbar-kiri/) and simpler asset management. Here is how these pages are structured in the workspace:

docs/
├── installation/
│   └── index.html  # Installation instructions
└── navbar-kiri/
    └── index.html  # Sidebar structure overview (this page)

Layout Highlights

This layout uses CSS Flexbox variables coupled with media queries to provide seamless transitions when navigating between folders. Key details include:

  • Sidebar Highlighting: The active list item in the left sidebar matches the active document routing folder path.
  • SEO Breadcrumbs: Dynamic breadcrumbs represent your folder levels to enhance SEO scoring.
  • Sticky Headers: The top navbar and sidebars utilize CSS sticky behaviors (position: sticky) for easier reading during long scrolls.
