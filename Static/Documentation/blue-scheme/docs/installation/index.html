Installation

v1.0.0

To use BlueDoc, you must have the following prerequisites installed on your system. Follow the steps below to configure your documentation platform.

Prerequisites

Please make sure you have the following environments configured before running BlueDoc:

  • node (v18.0.0 or higher for command utilities)
  • git (required for documentation hosting and clone hooks)
  • curl or wget (to fetch packages and installers)

Installation Steps

You can install BlueDoc directly through our global CLI script. Run the following command in your terminal:

# Download and execute the installation script
curl -fsSL https://bluedoc.dev/install.sh | sh
Note
If you encounter permission issues during script execution, make sure you have execution rights or run with administrative privileges where appropriate.

Verify Installation

After installation completes, check if the executable is accessible on your system path by querying its version:

# Check the installed version
bluedoc --version

If the terminal prints BlueDoc version 1.0.0, your installation was successful and you are ready to configure your documentation folders!
