About BlueDoc

A custom documentation layout design created with premium aesthetics and responsive light-blue accents.

BlueDoc is a demonstration web application showcasing clean web structures, custom layouts, and a responsive experience. This project replicates features inspired by modern open-source toolpages, specifically mirroring the sticky headers, clean sidebars, and theme-toggle configurations found in the reference layout.

Designed with HSL Colors

Utilizes dynamic HSL light-blue palettes to deliver rich aesthetics. Hover indicators and smooth transitions are configured via CSS variables.

Clean Codebase

No JSX, TypeScript, or heavy frameworks. Pure, readable, and highly maintainable vanilla HTML5, CSS3, and JavaScript.

Use the top navigation to explore the main homepage, browse documentation, or inspect layout behaviors by switching the global light/dark mode theme.
